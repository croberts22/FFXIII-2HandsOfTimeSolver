- (void) atIndex: (int)  index
withArray: (NSMutableArray *)  array
andSolution: (NSArray *)  current_solution
solveWithRemaining: (int)  remaining 

Recursively goes through the puzzle and determines the first possible solution.

The solver goes in a clock-wise position. The resulting array is saved in 'solutions' if a solution exists.

Parameters:
indexThe position in the puzzle that the solver is currently looking at within the recursion.
arrayAn array that contains the numbers in the puzzle. Solved numbers are replaced with a -1 value.
current_solutionAn array that contains the current solution (positional values) through the recursion.
remainingAn int that describes how many numbers are left to solve in the puzzle.
